# LabFoodExpress

## Introducción
LabFoodExpress es una aplicación que tiene como objetivo facilitar el acceso a comida de calidad a través de una plataforma digital.

## Características
- Acceso a los restaurantes más populares.
- Opciones de pago seguras.
- Seguimiento de pedidos en tiempo real.
